## Frontend (React + Vite)

Made using react+tailwind ,it uses router-dom for page transition as well as axios to communicate with the backend.

**Key Features** :
- Dynamic Routing: Powered by react-router-dom for seamless page transitions (Login, Signup, Dashboard, Trip Details).

- State Management: Local storage and React state synchronization to maintain user sessions.

- Utility-First Styling: Built with Tailwind CSS v4 for a clean, modern aesthetic.

- External Integration: Axios-based services to communicate with the backend and handle search queries.

**Tech Stack**
- Framework: React 18+ (Vite)
- Styling: Tailwind CSS v4
- Navigation: React Router 7
- API Client: Axios
